## Linkshortener

Link shortener is a website where you can manage short link.

## Install

### Support

- Ubuntu 20.04.2
- Windows 10/11
- MacOS

### Manual

Requirements:

- [npm](https://www.npmjs.com/)

## Usage

```bash
npm i
npm start
```

Be sure to have a .env file with the REACT_APP_BITLY_TOKEN variable in with your bitly token.
